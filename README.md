# 💱 Currency Converter Web App

## 📌 Overview

A modern and responsive **Currency Converter** built using **HTML, CSS, and JavaScript**.
This app allows users to convert currencies in real-time using exchange rate APIs with a clean UI and smooth user experience.

## 🚀 Live Demo

👉 https://kiran-Jathar.github.io/Currency-Converter/


## ✨ Features

* 🌍 Real-time currency conversion
* 💱 Supports multiple currencies
* 🔄 Swap currencies with one click
* 🌙 Dark mode toggle
* 📱 Responsive design (mobile + desktop)
* ⚡ Fast and lightweight
* 📋 Copy result to clipboard
* 🔔 Toast notifications for user feedback



## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript**
* **Exchange Rate API**
